# TheMessage
风声客户端
---
使用Cocos Creator v3.8.0

## 安装依赖包
`npm install`

## 生成portobuf
`npm run build:proto`

## windows平台
在Cocos Creator 编辑器按下列顺序执行
---
项目 -> 构建发布 -> 新建构建任务 -> web手机端
---
编辑器中构建完成后执行
---
`npm run build:electron`

## android和其他平台
根据[Cocos Creator官方文档](https://docs.cocos.com/creator/manual/zh/editor/publish/)操作
